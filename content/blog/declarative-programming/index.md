# 宣言型プログラミングとは
得ようとする出力の性質の宣言のみをコードに記述する手法。細かい処理は後のレイヤーに委譲される。

非命令型プログラニングと同義で扱われることが多い。

料理で例えると、調理人にカレー 1人前とメモを渡すイメージ。
このとき、調理人がコンピュータで、メモがコードである。

ニンジンを角切りにするか乱切りにするかは調理人に委譲される。

```
# レシピ
カレー 1人前
```
